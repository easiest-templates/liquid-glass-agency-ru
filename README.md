# Liquid Glass Agency (RU)

Современный лендинг для digital-агентства с эффектами стекломорфизма (glass morphism) и жидкими анимациями.

## Превью

![Liquid Glass Agency](https://cdn.poehali.dev/templates/liquid-glass-agency/preview.png)

## Особенности

- Glass morphism эффекты (стекломорфизм)
- Жидкие анимации фона (blob animations)
- Framer Motion анимации при скролле
- Адаптивный дизайн
- Темная тема
- Полностью на русском языке
- Шрифт Inter с поддержкой кириллицы

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- shadcn/ui компоненты

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

```
src/
  components/
    Navbar.tsx      # Навигация с glass-эффектом
    Hero.tsx        # Главный экран с blob-анимациями
    Services.tsx    # Секция услуг с glass-карточками
    Work.tsx        # Портфолио проектов
    Footer.tsx      # Подвал сайта
    ui/
      glass-card.tsx  # Компонент glass-карточки
  pages/
    Index.tsx       # Главная страница
  index.css         # Глобальные стили и CSS-анимации
```
